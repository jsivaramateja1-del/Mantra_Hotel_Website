# Royal Haven Hotel — Luxury Hotel Website

A complete frontend hotel website built for the **MANTRA 2026 Summer School Assignment**.

## Website Type
Hotel Website

## Hotel Name
**Royal Haven Hotel & Suites** — New Delhi, India

## Technologies Used
- HTML5 (semantic page structure)
- CSS3 (responsive layout, media queries, custom properties)
- JavaScript (form validation, gallery filter, cost calculator, FAQ toggle, mobile menu)
- GitHub (code hosting)
- Netlify (live deployment)

## Pages Included

| Page | File | Purpose |
|------|------|---------|
| Home | `index.html` | Hero banner, featured rooms, about snippet, testimonials, CTA |
| About | `about.html` | Hotel story, values, team, statistics |
| Rooms | `rooms.html` | All 6 room types with prices + cost calculator |
| Facilities | `facilities.html` | Spa, pool, dining, gym, FAQ toggle |
| Gallery | `gallery.html` | Filterable gallery (rooms/dining/spa/events/exterior) |
| Booking | `booking.html` | Validated room booking form |
| Contact | `contact.html` | Contact form, location, department contacts |

## JavaScript Features
1. **Mobile Menu** — Hamburger toggle for small screens
2. **Booking Form Validation** — Name, email, phone, date, room type checks
3. **Contact Form Validation** — Name, email, subject, message checks
4. **Room Cost Calculator** — Calculates total cost with 12% GST
5. **Gallery Filter** — Filter images by category (All/Rooms/Dining/Spa/Events/Exterior)
6. **FAQ Toggle** — Accordion expand/collapse on Facilities page
7. **Active Nav Link** — Highlights current page in navigation

## SEO Implementation
- Unique `<title>` tag on every page
- `<meta name="description">` on every page
- `<meta name="keywords">` on every page
- Single `<h1>` per page
- `<h2>` and `<h3>` for sections
- `alt` attributes on all images
- Semantic HTML (`<header>`, `<main>`, `<footer>`, `<article>`, `<section>`, `<nav>`)
- Meaningful file names (`rooms.html`, `facilities.html`, etc.)
- Internal links across all pages

## Responsive Design
- CSS custom properties (design tokens)
- CSS Grid with `@media` queries
- Breakpoints at 960px (tablet) and 640px (mobile)
- Flexible typography with `clamp()`

## How to Run Locally
1. Download all files into one folder
2. Open `index.html` in any web browser

## Netlify Deployment Steps
1. Create a public GitHub repository
2. Upload all project files
3. Log in to [Netlify](https://netlify.com)
4. Click **"Add new site"** → **"Import an existing project"**
5. Connect your GitHub repository
6. Click **Deploy** — site goes live in seconds!
